Module OgamlUtils.Interpolator

Interpolation between multiple values
This module defines interpolators between two values. An interpolator between value v0 and v1 is a function f such that f(0) = v0, f(1) = v1.
Type of an interpolator returning type 'a
Raised when an error occurs during the creation of an interpolator
