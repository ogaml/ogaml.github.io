Module OgamlGraphics.VertexMap

Customisable, high-level vertex arrays

Submodules

Vertex Represents a vertex
Source Represents a source of vertices
This modules provides a high-level and safe access to openGL vertex arrays. Vertex maps are less safe and optimized than vertex arrays, but can store any kind of data (especially integers). You should use the module VertexArray when possible.
