Module OgamlGraphics.VertexMap

Customisable, high-level vertex arrays

Submodules

Source Represents a source of vertices
Vertex Represents a vertex

This modules provides a high-level and safe access to openGL vertex arrays. Vertex maps are less safe and optimized than vertex arrays, but can store any kind of data (especially integers). You should use the module VertexArray when possible.

Raised when trying to rebuild a vertex map from an invalid source
Raised if a vertex passed to a source has a wrong type
Raised if an attribute defined in a GLSL program does not have a type matching the vertex's
Raised when trying to draw with a vertex map containing an attribute that has not been declared in the GLSL program
Raised when trying to draw an invalid slice of a vertex map
Phantom type for static maps
Phantom type for dynamic maps
Type of a vertex map (static or dynamic)
Creates a static map from a source. A static map is faster but cannot be modified later.
See : OgamlGraphics.VertexMap.Source
Creates a dynamic vertex map that can be modified later.
See : OgamlGraphics.VertexMap.Source
rebuild map src offset rebuilds map starting from the vertex at position offset using src .
The vertex map is modified in-place and is resized as needed.
See : OgamlGraphics.VertexMap.Source
Returns the length of a vertex map
Draws the slice starting at start of length length of a vertex map on a window using the given parameters.
start defaults to 0
if length is not provided, then the whole vertex map (starting from start ) is drawn
uniform should provide the uniforms required by program (defaults to empty)
parameters defaults to DrawParameter.make ()
mode defaults to DrawMode.Triangles
