Module OgamlGraphics.VertexArray.Source

Represents a source of vertices
This module provides a way to store vertices in a source before creating a vertex array. Note that a source is a mutable structure, therefore add and (<<) will directly modify the source. Sources are redimensionned as needed when adding vertices.
Type of a source
