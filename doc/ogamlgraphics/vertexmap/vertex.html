Module OgamlGraphics.VertexMap.Vertex

Represents a vertex
This module encapsulates vertices that can be passed to a source. A vertex is an immutable collection of attributes.
Type of a vertex
Empty vertex
Adds a vector3f to a vertex. The given name must match the name of the vec3 attribute in the GLSL program See : OgamlMath.Vector f
Adds a vector2f to a vertex. The given name must match the name of the vec2 attribute in the GLSL program See : OgamlMath.Vector f
Adds a vector3i to a vertex. The given name must match the name of the ivec3 attribute in the GLSL program See : OgamlMath.Vector i
Adds a vector2i to a vertex. The given name must match the name of the ivec2 attribute in the GLSL program See : OgamlMath.Vector i
Adds an integer to a vertex. The given name must match the name of the int attribute in the GLSL program
Adds a float to a vertex. The given name must match the name of the float attribute in the GLSL program
Adds a color to a vertex. The given name must match the name of the vec4 attribute in the GLSL program See : OgamlGraphics.Color
