Module OgamlGraphics.Uniform

Encapsulates a group of uniforms for rendering
This module encapsulates a set of uniforms that can be passed to GLSL programs
Raised when trying to draw using a program that requires a uniform not provided in the set
Raised when the type of a uniform is not matching the type required by the GLSL program
